What is "hard coding"?

When text is explictly written in the HTML, we say that it is hard coded. This means it can only be changed if someone has access to the site's files.

HTML


<h1>Fuzzy boots</h1>
<h3>$139.99</h3>
<p>These boots will keep you warm all winter. Their truly amazing!</p>

Consequences of hard coding: Difficuly to change if you don't know where it is; not everyone will know how to change it; sometimes you can't access it.

How does Liquid help?

Instead of writing out explicity what we want, we just reference an object.

Liquid


<h1>{{ product.title }}</h1>

<p>{{ product.description }}</p>

HTML returned


<h1>Rooster Decor Wood Cheese Board</h1>

<p>This is such a lovely wood grain rooster cheese board.
Great country / rustic decor for the rooster collector.</p>

If you want to change something, you just have to login into Shopify and change the page. No need to go into the theme files.

Exercise #4: Intro to liquid filters

  • You've decided you need to see the product's type listed after the product's description.
  • And you want to see the product type in ALL CAPS.

To accomplish the second task, we'll need to know what liquid filters are available to us.
Filters modify whatever they are attached to, whether numbers, strings, variables or objects

Example


{{ product.object | filter }}

There are two instances of product.title - need to choose the right one. Explain what a "pipe" is and where to find it. Put a picture of the keyboard and point out where it is.

Types of filters

String Filters: for when the variable type is a string.

Liquid


{{ page.title | upcase }}
{{ 'HOW ARE YOU?' | downcase }}
{{ product.handle | append: '.png' }}
{{ article.author | replace: 'Nathan', 'Bob' | prepend: 'Sir ' }}

Exercise #4: Intro to liquid filters

You now want the product's type to also be a link to a page with all products of that type.

This is totally possible. You just need the right type of filter on the string.

Liquid


{{ product.type | upcase | link_to_type }}

HTML


<a href="/collections/types?q=JEWELRY" title="JEWELRY">JEWELRY</a>

Exercise #5: Collection page hero images

By looking up what liquid objects are available to us for collections, show a hero image at the top of our collection pages.

After <div id="collection"> in the collection.liquid:

collection.liquid


<div class="row">
  <div class="span12 collection-hero" style="background-image: url({{ collection.image.src | img_url: '1024x1024' }});" >
    <h1 class="collection-title">{{ collection.title }}</h1>
  </div>
</div>

What is img_url: '1024x1024'? Check out Shopify's preset image size parameters.

At the bottom of styles.scss.liquid write:

styles.scss.liquid


.collection-hero{
  background-repeat: no-repeat;
  background-size: cover;
